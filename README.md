# Weather app using NodeJS

## Description
This is a CLI based weather application using NodeJS. The API's used are [darksy weather](https://darksky.net/dev) and Google Maps. Axios package simplified the code by using Promise based HTTP requests. The application retrieves the latitude and longitude based on the input address to fetch the weather information from darksky. 

## Getting Started
Packages required are **axios**, **yargs** and **request**. 
Example input in terminal: `node nameOfFile.js -a '350 Vermont St. Brooklyn'`.
